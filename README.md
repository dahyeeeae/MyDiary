# 🌟 2025-2 정보문화기술입문 기말 프로젝트 — My Diary

귀여운 스티커 붙이고, 프레임 골라서 꾸미고, 펜으로 끄적이고, 웹캠으로 사진도 “찰칵!”  
완성본은 QR 코드로 바로 가져갈 수 있는, 체험형 p5.js 웹 그림일기 프로젝트입니다. 🍪✨

---

### ✨ 주요 기능

🖼️ 프레임 선택: Todo 리스트/우주/리본/그림일기 등 4가지 테마  
🧸 스티커: 패널에서 드래그해 배치 (크기 조절·회전·삭제 가능)  
✏️ 그리기/지우개: 굵기·색상 선택 가능  
📸 웹캠: P 키로 즉시 캡처하여 캔버스에 삽입  
☁️ 업로드 & QR: 결과를 Firebase에 업로드 → 생성된 링크를 QR 코드로 표시
